# 重点分析标准

## 公司重点标准（2026年6月版）

### 核心指标

| 关注类型 | 筛选条件 | 优先级 |
|---------|---------|-------|
| **冲三档** | 当前标准2，离标准3（6h/39单）差 ≤ 2单 或 ≤ 20min | ⭐ 最高 |
| **全部升档机会** | 任一档位离升级差 ≤ 2单 或 ≤ 20min | ⭐ 高 |

### 差2单/20min 的含义

- **差2单**：gap_orders ≤ 2 且 > 0（差1单或2单就算）
- **差20min**：gap_hours ≤ 0.33（0.33h = 20分钟）

### 输出层级

```
1. 冲三档名单（最高优先级）
   → 骑手、实完单、实在线、差单、差时、升级条件

2. 全部升档机会
   → 骑手、当前档位、实完单、实在线、目标档位、差距

3. 汇总
   → 当前标准3以上人数
   → 冲标准3机会人数
   → 潜力可达人数（标准3以上 + 冲标准3可升者）
```

### 日报话术建议

- **结论需包含**：在不花钱的前提下，排班+话术能做什么
- 冲标准3骑手：可调整班次覆盖高峰时段，或当面沟通激励提升在线/单量
