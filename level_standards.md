# 档位标准（2026年6月版）

## 判定表

| 档位 | 要求在线 | 要求完单 | 标签 |
|------|---------|---------|------|

## 取低原则

骑手的最终档位 = min(按在线时长得的档位, 按完单量得的档位)

```
在线 7h 完单 30单
  时长 → 标准4（≥7h）
  单量 → 标准1（≥18单, <31单）
  最终 → 标准1（取低）
```

## 档位标签输出

- `标准1` ~ `标准5` — 对应运力标准等级
- `未达标 / 不足标准1` — 在线 < 6h 或 完单 < 18单
