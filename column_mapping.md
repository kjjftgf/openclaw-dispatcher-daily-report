# 列映射

aeolus 实时出勤 Excel 的列与脚本处理对照。

## 输入列（出勤 Excel 表头）

| 列名 | 脚本映射 | 用途 |
|------|---------|------|
| 站点名称 | `站点` | 站点标识、分组统计 |
| 姓名 / 骑手id | `姓名` / `骑手id` | 骑手标识 |
| 排班时段 / 排班状态 | `排班` | 是否排班（排除空值/休息/未排班/请假） |
| 工作状态 | `工作` | `上班`=在线，其他=小休/下线 |
| 时段 | `时段` | 当前行所属时段（排除"全天"） |
| 时段有效在线时长 | `时段时长` | 该时段实际在线小时数 |
| 时段完单量 | `时段完单` | 该时段实际完成订单数 |

## 时间解析

支持格式：
- `7小时42分钟` → 7.7（小时）
- `07:42` → 7.7
- `-` / 空 → 0
- `2` → 2（纯数值 = 小时数）

## 输出列

| 列 | 说明 |
|----|------|
| 时段单量达标 | 1=达标, 0=不达标 |
| 时段时长达标 | 1=达标, 0=不达标 |
| 是否达标 | 1=都达标, 0=任一不达标 |
| 时间差额(分钟) | 在线-要求，负值=差时 |
| 单量差额 | 完单-要求，负值=差单 |
